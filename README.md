## About
タスクを管理する簡単なアプリケーションサーバーです。


## Environment

```bash
typescript : 4.3.5
nestjs : 8.2.6
PostgreSQL : 14.2  
pg : 8.7.3  
@nestjs/typeorm : 8.0.2  
typeorm : 0.2.45  
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
